# ihsanuddinsalav

🔥 Personal website & portfolio, built from scratch using Next.js, TypeScript, Tailwind CSS, SWR and Prisma.

## Overview

This repository contains the source code for the **ihsanuddinsalav** project.
It is primarily developed using **TypeScript**.

- Personal website and portfolio
- Built with Next.js, TypeScript, Tailwind CSS, SWR, and Prisma.

## Technologies Used

- **Primary Language**: TypeScript
- Node.js/npm (for dependencies)
- Tailwind CSS
- TypeScript

## Setup and Installation

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (LTS recommended)
- npm or yarn

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/Abelion512/ihsanuddinsalav
   ```
2. Navigate to the project directory
   ```bash
   cd ihsanuddinsalav
   ```
3. Install NPM packages
   ```bash
   npm install
   ```
